# Advent of Code 2024

Some solutions for the [Advent of Code 2024](https://adventofcode.com/2024/) challenge.

## Setup

This repo uses [`uv`](https://docs.astral.sh/uv/).

```zsh
❯ uv python install 3.12.8
❯ uv venv
```

## Run

To run a solution, use the following command:

```zsh
❯ uv run day01.py
```
